 
# What does this project do?

## An AI-powered query logging API. Send it a question, it queries Gemini, persists both question and response to SQLite, and returns the full record


# Why did I build it?

## Built to demonstrate end-to-end backend development — REST API design, ORM-backed persistence, and external API integration patterns commonly used in AI engineering.


# Live URL: https://ai-query-logger-production.up.railway.app/docs

# Stack: 

## - Python
## - FastAPI
## - SQLModel + SQLite
## - Google Gemini API
## - Deployed on Railway


# How does someone test it?

## Visit /docs to access the interactive Swagger UI. Try the POST /queries endpoint with a JSON body like {"question": "What is FastAPI?"} to see the full request/response cycle
